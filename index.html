SQL Projects

Covid 19 Data Exploration

In this Covid Data Exploration project I was able to import a large data set from an online data source. Query certain tables and explore the data using SELECT, FROM, WHERE statements. ORDER BY to create columns that were in ascending or descending alignment. CAST / CONVERT certain columns into the correct type of data type in order to perform mathematical operations. Then finally using two seperate methods, create a TEMP TABLE and CTE in order to perform more operations on columns created within a SELECT statement.

Data Cleaning Nashville Housing

In this Data Cleaning project I used housing data from the city of Nashville, TN. Imported the raw data and did some light exploration to see what needed to be cleaned. From there I identified 6 different problem areas to solve and began the data cleaning process. Standardizing date formats with CONVERT, populating property address fields with self JOINs, seperating address fields from city and state using two methods (ALTER TABLE, PARSENAME), converting data type bit (0,1) to nvarchar (Yes, No), removing duplicate rows with a CTE surrounding a Window function, and finally deleting unused columns.

Python Projects

Data Cleaning Movies

In this Data Cleaning project I used Movie data from Kaggle to search for any meaningful correlations. I went through different best practices to check for missing data, remove null values, modify data types, and drop duplicates. From there I began to look for correlations within the data set. Using Matplotlib to plot intesting scatter plots I was able to also generate a linear regression line with the help of the Seaborn library. Correlation matrix heatmap also helped to visual correlations in the remaining data after changing objects to numerical values. And finally I plotted a linear regression scatter plot with marginal distributions to display the bell curve associated with our data set.

Network Bandwidth Monitor

While traveling the United States and using a mobile hotspot primarily for internet connection I ran into the problem of running out of available data over a month span. This Network Bandwidth Monitor proved a solution by tracking live statistics of bandwidth usage which allowed for monitoring of streaming, downloads, and background data usage.
